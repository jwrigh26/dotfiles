Buffers & When Things Get Weird
Open Files in Memory

Buffers are just “open files in memory”. Windows show them; tabs group windows.

:ls :buffers
List all buffers
Numbers on the left are buffer IDs. Use them with :b.
:b 2 :b#
Jump to buffer #2 / jump to previous buffer
:bn :bp
Next / previous buffer
:bd :bd!
Delete buffer (close file) / force delete
Window stays; only the buffer is closed.
:wa :qa :qa!
Write all / quit all / force quit all
When Vim complains:
E37: No write since last change ‚Üí either save with :w or close just this file with :bd / :bd! (or your mappings ‚ê£c / ‚ê£C).
If you see a “swap file exists” message and you’re sure no other Vim has it open, choosing d to delete the swap is usually safe.
Prefer ‚ê£c over ‚ê£q when you just want to close a file and keep Vim running.

Navigation: Find, Search & Jumps
Ninja Moves

Character find on the line

fx Fx
Jump to next / previous x on this line
e.g. f) to go to the next closing parenthesis.
tx Tx
Jump just before next / previous x
Great for changing inside parens: dt), etc.
; ,
Repeat last f/F/t/T forward / backward

Search

/pattern ?pattern
Search forward / backward
n N
Next / previous search result
* #
Search word under cursor forward / backward
Perfect for jumping through usages of a variable.

Jump list & matching

Ctrlo Ctrli
Jump back / forward in location history
Think “back” and “forward” in browser history.
%
Jump to matching (), [], {}
Combo idea: jump into a function with %, then use } to hop to the next block, and Ctrlo to bounce back.

Motion Recipes (Quick Examples)
Operator + Count + Motion

Pattern: [operator] [count] [motion]. You just type the keys in order, no spaces. Think LEGO pieces snapping together.

5w
Move 5 words forward
Type: 5 then w.
3b
Move 3 words backward
Type: 3 then b.
4e
Move to end of the 4th word ahead
Nice for landing exactly at the end of a token sequence.
d2w 2dw
Delete 2 words forward
Both forms work. Type: d 2 w or 2 d w.
d$
Delete from cursor to end of line
Type: d then $.
3dd
Delete 3 lines
Type: 3 then d then d.
df)
Delete up to and including next ) on this line
Type: d then f then ).
dt;
Delete up to (but not including) next ;
Type: d then t then ;.
ciw
Change inner word
Cursor anywhere in a word ‚Üí type: c i w.
ci]
Change inside square brackets [...]
Cursor anywhere inside ‚Üí type: c i ].
ci)
Change inside parentheses (...)
Cursor inside ‚Üí type: c i ).
ci"
Change inside double quotes
Cursor anywhere between quotes ‚Üí type: c i ".
Drill idea: pick one recipe (like ci]) and use it 10 times in a row instead of manually deleting and retyping. Let the muscle memory bake in.
